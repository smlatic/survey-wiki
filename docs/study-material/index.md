# Study Material

!!! tip "How to use"
    - **Quiz** -- Select an answer to see if you're correct, with detailed explanations
    - **Flashcards** -- Click to flip, use arrows to navigate
    - **Mind Map** -- Click nodes to expand or collapse branches
